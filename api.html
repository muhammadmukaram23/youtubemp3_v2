Quick Start Guide

Getting Started

Our API provides simple endpoints to convert YouTube videos to MP3 format. Follow these basic steps:

  1. Send a POST request to /convert with the YouTube URL
  2. Get the task ID from the response
  3. Check conversion status using /task/{task_id}
  4. Download the MP3 file when ready

Convert Video to MP3

Request

POST /convert

Body Parameters:

  • url - YouTube video URL
  • quality - Audio quality (low, medium, high, ultra)

Response

{
  "task_id": "abc123def456",
  "status": "processing"
}

Check Status & Download

Check Status

GET /task/{task_id}
Status values: processing, completed, failed

Download File

GET /download/{task_id}
Note: Only available when status is "completed"

Error Handling & Limits

Common Status Codes

200 Success
400 Bad request
500 Server error

Rate Limits

  • • 100 requests/hour per IP
  • • 10 concurrent conversions
  • • Max 1 hour video duration
